# Colombia Staking

Connect Colombia Staking to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/colombia-staking) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-colombia-staking@joai-claude-plugins
```

### Codex

```bash
codex mcp add joai-colombia-staking --url https://cortex.joai.ai/mcp/apps/colombia-staking
```

### Cursor

Copy to local plugins:

```bash
git clone https://github.com/joai-plugins/colombia-staking.git ~/.cursor/plugins/local/joai-colombia-staking
```

Or use the [MCP deeplink](cursor://anysphere.cursor-deeplink/mcp/install?name=joai-colombia-staking&config=eyJtY3BTZXJ2ZXJzIjp7ImpvYWktY29sb21iaWEtc3Rha2luZyI6eyJ1cmwiOiJodHRwczovL2NvcnRleC5qb2FpLmFpL21jcC9hcHBzL2NvbG9tYmlhLXN0YWtpbmcifX19) to install directly.

### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/colombia-staking
```
